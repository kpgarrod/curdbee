# CurdBee

Ruby wrapper for the CurdBee API

## Installation

sudo gem install curdbee
    
## Examples

Check `examples/` directory for example usage.

## How to Contribute? 
 
    $ git clone git://github.com/vesess/curdbee.git
    $ cd curdbee
    $ bundle install
    $ bundle exec rake
